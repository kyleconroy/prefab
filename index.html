prefab - the basic configuration manager

prefab is a configuration manager for single machines. It configures machines in a simple and idempotent way. It is a standalone binary and consumes JSON. It is not meant to be a replacement for a full-fledged solution like Chef, Puppet, Ansible or Salt. It's ideal for creating reproducible development VMs.

$ wget https://stackmachine.com/pf.tgz
$ untar pf.tgz
$ echo '{"file":"/etc/foobar"}' >> foobar.json
$ pf apply foobar.json

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

prefab with Vagrant

prefab doesn't need a special provisioner, just invoke it using the shell provisioner like this:

$ provisioner :shell, :inline => 'pf apply node.json'

Contribute

prefab is somewhere between the proof of concept stage and the alpha stage. But we're finding it useful. It's 100% written in Go, which makes packaging and distribution easy.
